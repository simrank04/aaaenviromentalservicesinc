📁 Project Description:
-----------------------------------------------------------------
This is a static website developed for AAA Environmental Services Inc, 
a professional environmental cleanup and service provider. The site 
includes key sections like About Us, Services, Contact, and a 
responsive footer with contact info and company logo.

-----------------------------------------------------------------
🛠️ Technologies Used:
-----------------------------------------------------------------
- HTML5: Markup structure.
- CSS3: Styling and responsive layout.
- JavaScript: Basic interactivity (e.g., navigation, animations).
- Bootstrap : For grid layout and responsive design.

-----------------------------------------------------------------
